# Long Division Calculator

A free, fast, and user-friendly online long division calculator with step-by-step solutions. Built with vanilla JavaScript, HTML5, and CSS3 for optimal performance and SEO.

## Features

- **Step-by-Step Solutions**: See detailed breakdown of each division step
- **Decimal Support**: Handle both whole numbers and decimals
- **Remainder Display**: Shows both remainder and decimal formats
- **Calculation History**: Automatically saves your recent calculations
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Offline Support**: PWA capabilities for offline usage
- **SEO Optimized**: Structured data, semantic HTML, optimized meta tags
- **Fast & Lightweight**: No frameworks, pure vanilla JavaScript

## Tech Stack

- HTML5 (Semantic markup)
- CSS3 (Modern responsive design)
- Vanilla JavaScript (No dependencies)
- Service Worker (PWA/Offline support)
- Schema.org structured data
- Open Graph meta tags

## Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Start calculating!

Or simply visit: [https://divisioncalculator.sbs](https://divisioncalculator.sbs)

## Deployment

### GitHub Pages

1. Fork this repository
2. Go to Settings > Pages
3. Select main branch as source
4. Your site will be published at `https://yourusername.github.io/repository-name/`

### Cloudflare Pages

1. Push code to GitHub
2. Log in to Cloudflare Dashboard
3. Go to Pages > Create a project
4. Connect your GitHub repository
5. Deploy with default settings
6. Optional: Add custom domain
